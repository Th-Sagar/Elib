## Features

- Login and Register user
- Create, Read, Update, Delete data from the server

# Prerequisites

- Node.js 14+
- MongoDB
- VS-Code Extensions (Prettier, - ESLint, etc)
- Command Line Tools

## Deployment

To deploy this project run

```bash
 git clone https://github.com/Th-Sagar/Elib.git
```

```bash
cd Elib
```

```bash
npm i
```
